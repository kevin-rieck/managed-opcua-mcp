## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues for `kevin-rieck/managed-opcua-mcp`; external PRs are not a triage request surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the default triage label vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a single-context domain docs layout: root `CONTEXT.md` and `docs/adr/`. See `docs/agents/domain.md`.
